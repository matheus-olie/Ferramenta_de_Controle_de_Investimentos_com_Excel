# Ferramenta_de_Controle_de_Investimentos_com_Excel
Desafio realizado no Bootcamp Santander, pela DIO. A atividade requeria a criação de uma ferramenta de controle de investimentos para fundos imobiliários, de acordo ao perfil do investidor.

------------#-----------------------#---------------------------#-------------------------#

Foram utilizados as seguintes técnicas:
* Tabela Referencial
* Simulador de Patrimônio
* Simulador de Cenários
* Intervalos nomeados
* Variaveis Globais
* Uniformidade Visual
